# Landing Page - Entreprenly

Una landing page minimalista y ligera construida con HTML, CSS y Tailwind CSS.

## Instalación

Instala las dependencias:

```bash
npm install
```

## Desarrollo

Inicia el modo desarrollo:

```bash
npm run dev
```

## Compilación

Compila los estilos para producción:

```bash
npm run build
```

## Estructura del Proyecto

```
.
├── index.html          # Página principal
├── app.js              # JavaScript
├── styles.css          # Estilos compilados
├── package.json        # Configuración
├── .gitignore          # Archivos ignorados
├── assets/             # Imágenes y recursos
└── src/
    └── input.css       # Estilos Tailwind fuente
```
